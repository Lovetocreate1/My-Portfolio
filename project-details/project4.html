🚗 Project 4: Car Animation

🛣️ This project features a smooth and visually engaging car animation built with HTML, CSS, and JavaScript. The car glides across the screen with a dynamic looping road background, making it fun to watch and interact with.

✨ Highlights:

  • 🚘 Car moves fluidly across the screen
  • 🌄 Animated background simulates a moving road
  • 📱 Responsive on all screen sizes
  • 🎨 Created with pure CSS & JavaScript effects

🛠️ Tech Stack:

  • 🌐 HTML, CSS, JavaScript
  • ⏩ Keyframe animations for smooth transitions
  • 📐 Flexbox layout for responsiveness
  • 🚀 Hosted on Netlify
