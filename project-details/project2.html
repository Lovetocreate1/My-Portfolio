📝 Project 2: Personal Blog

  • 🔐 User authentication and login system

🛠️ Technologies Used:

  • 🔤 HTML, 🎨 CSS, and ⚙️ JavaScript
  • ⚡ React.js for dynamic content
  • 🔒 Firebase for authentication
  • 🚀 Deployed on Netlify
