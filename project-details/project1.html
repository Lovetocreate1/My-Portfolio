🛍️ E-Commerce Website

This project is a modern and fully responsive e-commerce web application built to deliver an engaging shopping experience.

✨ Features:

  • 🖥️ Modern UI with clean layout
  • 📦 Product listing and category filters
  • 🛒 Shopping cart with local storage
  • 👤 User login & registration
  • 🔐 Secure checkout with payment gateway

🛠️ Technologies Used:

  • 🌐 HTML, CSS, JavaScript
  • 🎨 Bootstrap for responsive design
  • 💾 LocalStorage for cart management
