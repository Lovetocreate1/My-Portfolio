🐉 Project 5: Dragon Game

🔥 Welcome to the Dragon Game — a thrilling and interactive browser-based game where you guide a flying dragon through obstacles, breathe fire, and enjoy epic gameplay with sound and animations.

✨ Key Features:

  • 🕹️ Touch & keyboard controls for all devices
  • 🎞️ Smooth flying and fireball animations
  • 💀 Game-over logic with instant restart
  • 🎧 Immersive background music and effects

🛠️ Tech Stack:

  • 🌐 HTML, CSS, JavaScript
  • 🎵 Custom audio, sprite sheets, and key events
  • 📱 Fully responsive for mobile & desktop
  • 🚀 Deployed on Netlify
